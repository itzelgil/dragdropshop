In this assignment, I created a digital style shop with the function "Drag and Drop"

Here you have the steps to start using it:

1. Select your season style
2. Look for the outfit that you like the most
3. Select it and drag it to the shopping zone
4. If you want, you can delete some items by clicking on the "x"
